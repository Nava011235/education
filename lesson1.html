DOM-структура

DOM - ( document object model ), объектная модель документа. DOM предостовляет js возможность взаимодействовать с элементами страницы ( div, input, form, etc). Все элементы страницы в DOM представляются, как обычные js объекты со своими методами, прототипами и свойствами Группа элементов (например все div-ы) могут представлять html-коллекции ( почти как массив, но не совсем). DOM создается при загрузки страницы, тут есть одно но!, модель создется постепенно ( грубо говоря сверху вниз ) поэтому js не может обратиться к DOM-узлу ( DOM можно представлять, как дерево, где каждый элемент это DOM-узел) , пока он не будет создан, напрмер

//Например у нас есть такая html страничка
‹body›
  ‹div id="firstDiv"› ‹/div›
‹/body›

И мы хотим иметь доступ к элементу div с id "firstDiv". DOM предоставляет нам метод объекта document - getElementById(id), возвращает DOM-узел с id, указанным в качестве параметра, либо NULL, если элемента с таким id ну существует. Теперь попробуем получить наш див

‹script›
  var div = document.getElementById("firstDiv");
  console.log(div) // вернет null
‹/script›
  ‹div id="firstDiv" › ‹/div›
‹script›
  var div = document.getElementById("firstDiv");
  console.log(div) // вернет объект нашего div-a
‹/script›

Почему же в первом случае код возращает NULL, во втором же случае возвращет уже готовый объект? Суть как раз в том, что на момент выполнения первого скрипта, div-a с id firstDiv еще не существует ( бразуер грубо говоря еще не дошел до него) следовательно и в DOM-дереве такого узла нет. Второй же раз, скрипт вызывается, когда в DOM-дереве уже есть нужный узел. Поэтому код возвращает нужный нам объект. DOM предостовляет нам несколько методов, для поиска объектов:

// Поиск по id элемента
  var idElement = document.getElementById("id")
// Поиск по классу элемента. Возвращает html-коллекцию
  var classElements = document.getElementsById("class")
// Поиск по тегу ( div, input, li )
  var tagElements = document.getElementsByTagName("tag")
// Поиск по css-селектору
  var queryElement = document.querySelector("selector")
